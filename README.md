# PCS Attendance System 🚀

## ✨ Starting

using npm:

```bash
npm install
```

or if you using yarn:

```bash
yarn install
```

> Create new env

## 🐢 Run

### Local

> for development in local

```bash
npm start
```

## 🍔 Build

Deploy

```bash
npm run build
```
